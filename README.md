# 如何安装

## 第一步

执行 NPM 包安装，并启动 Postgres 数据库

> yarn install

必须使用 yarn ！！！必须使用 yarn ！！！必须使用 yarn ！！！

## 第二步

执行数据库配置安装：

> yarn run:install

## 第三步

运行

> yarn run dev
